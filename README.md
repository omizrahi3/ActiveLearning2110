# ActiveLearning 2110

A Junior Design Project at Gatech


## Requirements

- [Node and npm](http://nodejs.org)
- [MongoDB](https://mongodb.com)

## Installation

Clone the repository:
```bash
git clone https://github.com/omizrahi3/ActiveLearning2110.git
```
Install the application:
```bash
npm install
```
Start MongoDB:
```bash
mongod
```
Start the server:
```bash
node server.js
```
View in browser at
```bash
http://localhost:8081
```
